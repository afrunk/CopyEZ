🔍 搜索功能

核心功能

CopyEZ 提供强大的实时搜索功能，支持标题、内容、标签的全文搜索，搜索结果以精美的卡片形式展示。

功能特点

  • 实时搜索：输入关键词后自动搜索（500ms 防抖）
  • 多字段搜索：同时搜索标题、内容、标签
  • 关键词高亮：搜索结果中关键词以红色高亮显示
  • 智能跳转：点击搜索结果自动跳转到详情页并定位到关键词
  • 空状态提示：未找到结果时显示「实事求是」书法背景

➕ 新增文章功能

核心功能

点击导航栏的「新建素材」按钮，可以快速创建新的素材文章。支持完整的分类体系、标签管理和内容编辑。

表单字段

  • 标题：文章标题（必填）
  • 一级分类：全文、框架、短文摘要（下拉选择）
  • 二级分类：讲话精神、调研报告等（可输入新分类，自动添加）
  • 三级标签：党建、政法、经济等（多选标签，支持搜索）
  • 发布日期：可选，用于记录文章发布时间
  • 原文链接：可选，保存原文 URL
  • 内容：支持 Markdown 格式，使用 #、##、### 表示标题层级

智能功能

  • 深度格式清洗：自动去除行首空格，规范化换行符
  • 自动分类管理：输入新的二级分类时自动添加到数据库
  • 标签搜索：输入标签时支持搜索和快速选择
  • 数据校验：标题和内容为空时显示错误提示

使用流程

  1. 点击导航栏右上角的「➕」按钮
  2. 填写标题和内容（必填）
  3. 选择或输入分类和标签（可选）
  4. 点击「保存」按钮完成创建
  5. 自动跳转到新创建的文章详情页

技术实现

  • 路由：GET/POST /new
  • 模板文件：templates/new.html
  • 标签选择器：static/category-selector.js
  • 内容清洗：deep_clean_content() 函数
  • 数据存储：SQLite 数据库，自动生成创建时间

✏️ 批注与笔记

交互功能

在阅读素材时，选中文本即可添加批注、标注，支持加粗、标蓝、标红等多种样式，批注会显示在侧边栏中。

使用流程

  1. 在详情页选中任意文字
  2. 弹出工具栏（包含：加粗、标蓝、标红、写批注、取消、复制）
  3. 选择标注样式或点击「写批注」添加笔记
  4. 批注显示在右侧侧边栏的「批注笔记」区域
  5. 点击已标注的文字可再次编辑或取消标注

🏷️ 三级分类筛选

组织功能

CopyEZ 采用三级分类体系组织素材：一级分类、二级分类、三级标签。点击任意标签即可快速筛选。

分类层级

  • 一级分类：全文、框架、短文摘要等（深色标签）
  • 二级分类：讲话精神、调研报告等（紫色标签）
  • 三级标签：党建、政法、经济等（浅色标签）
  • 动态筛选：点击标签后显示筛选状态条，可一键清除

分类标签展示

技术实现

  • URL 参数：?mainCategory=xxx&subCategory=xxx&tag=xxx
  • 标签样式：.tag-level-1、.tag-level-2、.tag-level-3
  • 筛选逻辑：基于 SQLAlchemy 查询过滤
  • 状态持久化：筛选条件保留在 URL 中

📋 首页列表展示

核心功能

首页以扁平化卡片列表展示所有素材，按创建时间倒序排列。支持点击标签快速筛选，鼠标悬停显示删除按钮。

列表展示

  • 卡片式布局：每条素材以白色卡片形式展示
  • 时间分组：按创建日期分组，显示「今天」「昨天」「更早」等
  • 关键信息：标题、分类标签、创建时间一目了然
  • 悬停效果：鼠标悬停时卡片显示阴影，提升可点击感
  • 空状态：无素材时显示「实事求是」书法背景

标签筛选

  • 点击标签：点击任意分类标签即可快速筛选
  • 筛选状态条：筛选后显示状态条，显示当前筛选条件
  • 一键清除：点击「清除筛选」按钮恢复全部列表
  • URL 参数：筛选条件保留在 URL 中，支持分享和刷新
  • 三级筛选：支持一级分类、二级分类、三级标签的独立或组合筛选

删除功能

  • 悬停显示：鼠标移动到卡片上时，右上角显示垃圾桶图标
  • 确认对话框：点击删除按钮弹出确认对话框，防止误删
  • 淡出动画：删除成功后卡片淡出并移除
  • 自动刷新：删除后自动检查列表，如果为空则显示提示
  • 错误处理：删除失败时显示错误提示

🔗 知识图谱推荐

智能功能

基于标签和分类的相似度匹配，自动推荐相关素材。支持任务链识别，如「方案」→「总结/通报/调研报告」。

推荐逻辑

  • 标签匹配：相同标签 +10 分（权重更高）
  • 分类匹配：相同二级分类 +5 分
  • 任务链识别：检测标题中的「方案/计划/行动」关键词
  • 时间轴展示：关联素材以时间轴形式展示
  • 数量限制：最多推荐 5 篇相关素材

🔒 锁定屏幕

安全功能

点击导航栏的锁定按钮，会显示「求真务实」书法门扉，需要输入密码解锁。密码存储在本地。

功能特点

  • 书法背景：「实事求是」书法图片作为锁定背景
  • 幕布动画：上下幕布合拢效果
  • 密码保护：输入正确密码解锁
  • 本地存储：密码存储在 localStorage 中
  • 一键锁定：点击导航栏锁定按钮即可锁定

📌 侧边栏收起

交互优化

详情页侧边栏右侧有一个现代化的长条拉环，支持收起/展开操作，状态会持久化保存。

交互细节

  • 长条拉环：固定在侧边栏左边缘
  • 智能显示：鼠标靠近屏幕右侧 150px 时显示
  • 自动隐藏：3 秒未移动自动隐藏
  • 状态持久化：收起/展开状态保存在 localStorage
  • 平滑动画：CSS transition 实现平滑过渡

技术实现

  • 组件类：.sidebar-collapse-toggle
  • 状态键：sidebarCollapsed（localStorage）
  • 触发区域：屏幕右侧 150px 范围内
  • 动画时长：300ms transition

📝 编辑器样式

展示功能

CopyEZ 支持丰富的富文本样式，包括不同层级的标题、加粗、颜色标注等，符合公文写作规范。

支持的样式

  • 标题层级：使用 #、##、### 生成 h1、h2、h3
  • 字体规范：一级标题（黑体）、二级标题（楷体）、三级标题（仿宋）
  • 加粗标注：选中文字后点击「加粗」按钮
  • 颜色标注：支持蓝色（标蓝）和红色（标红）
  • A4 纸张：详情页采用 A4 纸张样式，支持打印

📄 公文样式模板

展示功能

CopyEZ 严格按照 GB/T 9704-2012《党政机关公文格式》国家标准设计，提供完整的公文排版样式模板。

A4 纸张规范

  • 纸张尺寸：210mm × 297mm（A4 标准）
  • 页边距：上 3.7cm、右 2.8cm、下 3.5cm、左 2.6cm
  • 打印支持：打印时自动应用 A4 纸张和页边距设置
  • 阴影效果：屏幕显示时添加纸张阴影，提升阅读体验

字体规范

  • 总标题：方正小标宋简体，2 号（22pt），居中，不缩进
  • 一级标题：黑体，16pt（21px），左对齐，缩进 2em
  • 二级标题：楷体_GB2312，16pt（21px），左对齐，缩进 2em
  • 三级标题：仿宋_GB2312，16pt（21px），左对齐，缩进 2em
  • 正文：仿宋_GB2312，16pt（21px），行距 1.56 倍，段首缩进 2em

排版规范

  • 段落缩进：所有段落统一使用 CSS text-indent: 2em，禁止手动空格
  • 行间距：固定倍数 1.56（对应 28.95mm 标准行距）
  • 段落间距：0（仅依靠 line-height 控制垂直节奏）
  • 标题间距：紧凑型排版，减小上下边距
  • 空行处理：自动合并多个连续空行

⚙️ 技术参数

开发文档

部署环境

  • 框架：Flask（Python）
  • 数据库：SQLite（copyez.db）
  • 存储路径：本地文件系统
  • 静态资源：static/ 目录
  • 字体文件：fonts/ 目录（方正字体、楷体、仿宋）

核心样式类

  • .hl-bold - 加粗标注
  • .hl-blue - 蓝色标注
  • .hl-red - 红色标注

  • .heading.level-1/2/3 - 标题层级
  • .tag-level-1/2/3 - 分类标签

API 端点

  • GET /api/search?q=关键词 - 搜索素材
  • GET /api/note/{id}/related - 获取关联素材
  • POST /api/note/{id}/annotations - 添加批注
  • DELETE /api/note/{id}/annotations/{ann_id} - 删除批注
