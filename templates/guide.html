CopyEZ 功能指南

从「平时积累」到「随手可用」，这页帮你快速理解 CopyEZ 的核心能力：公文素材库、沉浸式阅读、随心记备忘，一张工作台全部搞定。

🔍 搜索与高亮

核心功能

CopyEZ 提供「长文 Note + 随心记 Memo」双轨制搜索：顶部搜索框统一入口，结果页以双 Tab 形式分别展示正式文章和随心记，所有命中片段都会高亮关键词，便于扫读。

功能特点

  • 统一入口：导航栏搜索框直接跳转到搜索结果页，支持再次编辑关键字
  • 双轨结果：页面上方「正式文章 / 随心记」Tab，可一键切换两类结果
  • 多字段搜索：正式文章按标题、正文内容、标签（含 JSON 标签）综合匹配
  • 随心记搜索：Memo 支持内容 + #标签 联合检索，并在结果中展示标签胶囊
  • 关键词高亮：所有结果里的命中词统一用 .search-highlight 做高亮，方便快速扫读
  • 智能跳转：点击正式文章结果在新标签页打开详情，并携带 ?highlight=关键词 便于二次定位

技术实现

  • 搜索 API：GET /api/search?q=关键词，返回 { articles: [], memos: [] }
  • 结果页路由：GET /search?q=关键词，模板 templates/search.html
  • 高亮样式：.search-highlight 类（背景高亮，用于标题、摘要和随心记内容）
  • 详情跳转：正式文章跳转链接为 /note/{id}?highlight=关键词，在新标签页打开

➕ 新增文章功能

核心功能

点击导航栏的「新建素材」按钮，可以快速创建新的素材文章。支持完整的分类体系、标签管理和内容编辑。

表单字段

  • 标题：文章标题（必填）
  • 一级分类：全文、框架、短文摘要（下拉选择）
  • 二级分类：讲话精神、调研报告等（可输入新分类，自动添加）
  • 三级标签：党建、政法、经济等（多选标签，支持搜索）
  • 发布日期：可选，用于记录文章发布时间
  • 原文链接：可选，保存原文 URL
  • 内容：支持 Markdown 格式，使用 #、##、### 表示标题层级

智能功能

  • 深度格式清洗：自动去除行首空格，规范化换行符
  • 自动分类管理：输入新的二级分类时自动添加到数据库
  • 标签搜索：输入标签时支持搜索和快速选择
  • 数据校验：标题和内容为空时显示错误提示

使用流程

  1. 点击导航栏右上角的「➕」按钮
  2. 填写标题和内容（必填）
  3. 选择或输入分类和标签（可选）
  4. 点击「保存」按钮完成创建
  5. 自动跳转到新创建的文章详情页

技术实现

  • 路由：GET/POST /new
  • 模板文件：templates/new.html
  • 标签选择器：static/category-selector.js
  • 内容清洗：deep_clean_content() 函数
  • 数据存储：SQLite 数据库，自动生成创建时间

✏️ 批注与笔记

交互功能

在阅读素材时，选中文本即可添加批注、标注，支持加粗、标蓝、标红等多种样式，批注会显示在侧边栏中。

使用流程

  1. 在详情页选中任意文字
  2. 弹出工具栏（包含：加粗、标蓝、标红、写批注、取消、复制）
  3. 选择标注样式或点击「写批注」添加笔记
  4. 批注显示在右侧侧边栏的「批注笔记」区域
  5. 点击已标注的文字可再次编辑或取消标注

🖼️ 沉浸式阅读体验

阅读增强

在详情页（公文馆阅读页）中，对图片阅读体验做了三件事：点击图片丝滑放大预览、进入视口时柔和淡入、自动显示图注，方便你拆解图表和动作细节。

图片交互

  • 点击放大：正文中的图片支持「Medium 风格」点击放大，再次点击或滚动即可还原
  • 丝滑动画：放大时背景轻微变亮，过渡平滑，不弹出生硬的弹窗
  • 不改排版：严格保留仿宋三号正文、2em 缩进和图片居中 / 宽度限制

懒加载淡入

  • 懒加载：长文中的图片默认懒加载，只有滚动到附近才会真正加载
  • 淡入效果：配合 .lazyload/.lazyloaded 类，图片会以 0.4s 的淡入动画出现
  • 性能友好：减少首屏加载压力，适配 2 核 2G 小服务器

自动图注

  • Markdown 直出：使用 ![图注文字](/static/uploads/xxx.png) 书写时，alt 文本会自动转为图下注释
  • 样式统一：图注以浅灰、小号、居中显示在图片下方
  • 纯前端增强：不修改 Markdown 解析逻辑和图片白名单，仅在前端追加说明文字

🏷️ 三级分类筛选

组织功能

CopyEZ 采用三级分类体系组织素材：一级分类、二级分类、三级标签。点击任意标签即可快速筛选。

分类层级

  • 一级分类：全文、框架、短文摘要等（深色标签）
  • 二级分类：讲话精神、调研报告等（紫色标签）
  • 三级标签：党建、政法、经济等（浅色标签）
  • 动态筛选：点击标签后显示筛选状态条，可一键清除

  • 标签样式：.tag-level-1、.tag-level-2、.tag-level-3

  • 状态持久化：筛选条件保留在 URL 中

📋 首页列表展示

核心功能

首页以扁平化卡片列表展示所有素材，按创建时间倒序排列。支持点击标签快速筛选，鼠标悬停显示删除按钮。

列表展示

  • 卡片式布局：每条素材以白色卡片形式展示
  • 时间分组：按创建日期分组，显示「今天」「昨天」「更早」等
  • 关键信息：标题、分类标签、创建时间一目了然
  • 悬停效果：鼠标悬停时卡片显示阴影，提升可点击感
  • 空状态：无素材时显示「实事求是」书法背景

标签筛选

  • 点击标签：点击任意分类标签即可快速筛选
  • 筛选状态条：筛选后显示状态条，显示当前筛选条件
  • 一键清除：点击「清除筛选」按钮恢复全部列表
  • URL 参数：筛选条件保留在 URL 中，支持分享和刷新
  • 三级筛选：支持一级分类、二级分类、三级标签的独立或组合筛选

删除功能

  • 悬停显示：鼠标移动到卡片上时，右上角显示垃圾桶图标
  • 确认对话框：点击删除按钮弹出确认对话框，防止误删
  • 淡出动画：删除成功后卡片淡出并移除
  • 自动刷新：删除后自动检查列表，如果为空则显示提示
  • 错误处理：删除失败时显示错误提示

🔗 知识图谱推荐

智能功能

推荐逻辑

  • 标签匹配：相同标签 +10 分（权重更高）
  • 分类匹配：相同二级分类 +5 分
  • 任务链识别：检测标题中的「方案/计划/行动」关键词
  • 时间轴展示：关联素材以时间轴形式展示
  • 数量限制：最多推荐 5 篇相关素材

📄 公文样式模板

展示功能

CopyEZ 严格按照 GB/T 9704-2012《党政机关公文格式》国家标准设计，提供完整的公文排版样式模板。

A4 纸张规范

  • 纸张尺寸：210mm × 297mm（A4 标准）
  • 页边距：上 3.7cm、右 2.8cm、下 3.5cm、左 2.6cm
  • 打印支持：打印时自动应用 A4 纸张和页边距设置
  • 阴影效果：屏幕显示时添加纸张阴影，提升阅读体验

字体规范

  • 总标题：方正小标宋简体，2 号（22pt），居中，不缩进
  • 一级标题：黑体，16pt（21px），左对齐，缩进 2em
  • 二级标题：楷体_GB2312，16pt（21px），左对齐，缩进 2em
  • 三级标题：仿宋_GB2312，16pt（21px），左对齐，缩进 2em
  • 正文：仿宋_GB2312，16pt（21px），行距 1.56 倍，段首缩进 2em

排版规范

  • 段落缩进：所有段落统一使用 CSS text-indent: 2em，禁止手动空格
  • 行间距：固定倍数 1.56（对应 28.95mm 标准行距）
  • 段落间距：0（仅依靠 line-height 控制垂直节奏）
  • 标题间距：紧凑型排版，减小上下边距
  • 空行处理：自动合并多个连续空行

碎片记录

除了长文素材库，还有一个专门记录「一闪而过想法」的随心记面板，支持 #标签 自动高亮与检索，不打扰你的长文体系。

记录体验

  • 快捷输入：顶部输入框支持多行文本，按 Ctrl + Enter 快速保存
  • 实时追加：新记录会立即出现在下方时间轴列表中
  • 空状态提示：没有记录时展示友好的引导文案

#标签系统

  • 自动提取：在内容中输入 #专项行动、#调研 等，保存时自动提取为标签
  • 高亮显示：标签在卡片中以醒目的样式高亮，便于扫读
  • 后续检索：标签字段会持久存入数据库，支持后续按标签检索与统计

技术实现

  • 页面路由：GET /memos
  • API：GET/POST /api/memos、PUT/DELETE /api/memos/<id>

  • 前端高亮：.memo-tag 样式与高亮逻辑在 templates/memos.html 内联脚本中实现

🚀 编辑效率增强

体验优化 · 新增

针对日常「边看边改」的公文场景，CopyEZ 在编辑页做了一些小而高频的优化：底部固定操作坞、图片即粘即传、配合输入法的全局输入模式，让你少动鼠标，多动键盘。

底部固定操作坞

  • 始终贴底：编辑页的「保存 / 取消」操作条固定在浏览器底部，不再随着页面高度乱跳
  • 滚动友好：长文编辑时，无论滚多长，到达底部都能第一时间看到保存按钮
  • 留白保护：正文区域自动预留底部空间，避免最后几行被按钮遮住

图片「即粘即传」

  • 直接粘贴：从微信、QQ、Word 等复制图片后，直接在编辑器里按 Ctrl + V
  • 自动上传：后台调用 POST /api/upload_image 接口，将图片保存到 /static/uploads/
  • 自动插图：返回的地址自动写入 Markdown，形成 ![](/static/uploads/xxx.png) 语法
  • 格式白名单：当前支持 png / jpg / jpeg / gif / webp

全局输入模式配合

  • 输入法浮层：当你启用输入法的「全屏 / 全局输入」时，编辑器会尽量避免被其遮挡关键按钮
  • 推荐用法：长文建议分段输入：先在全局输入里写一小段，再回到页面继续滚动和编辑
  • 安全兜底：真正的数据仍保存在网页编辑器中，避免误关输入法导致内容丢失

技术实现

  • 操作坞样式：.edit-actions-dock 固定定位于视口底部
  • 图片上传 API：POST /api/upload_image，使用 allowed_image() 校验扩展名
  • 图片目录：static/uploads/（自动创建，不会覆盖旧文件）

📱 移动端适配

体验优化

虽然 CopyEZ 主要面向桌面浏览器，但在手机和平板上同样做了基础适配：保证能「看得清」「点得准」「不乱跳」，方便临时查阅与简单编辑。

布局与断点

  • 自适应宽度：内容区域在小屏设备上自动铺满屏幕，避免左右出现大面积留白
  • 导航简化：窄屏时顶部导航栏收紧，只保留核心入口按钮
  • 侧边栏折叠：阅读页的右侧批注栏在小屏幕上自动隐藏，只保留正文，必要时再展开

字号与点击区域

  • 阅读字号：正文在移动端略微放大，保证在 5–6 英寸手机上仍然可读
  • 按钮尺寸：常用操作（保存、删除、筛选等）在移动端保证至少 40px 的点击高度
  • 标签换行：分类标签在小屏下自动换行，避免横向滚动

交互细节

  • 滚动容差：长文阅读时，底部操作坞不会遮挡最后几行正文
  • 输入体验：在手机键盘弹出时，编辑区域自动上移，尽量避免「只剩半行」的极端情况
  • 锁屏页面：锁定屏幕的「求真务实」书法在窄屏上等比例缩放，保证完整展示
