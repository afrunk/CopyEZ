# Changelog - CopyEZ

所有关于本项目的显著更新都将记录在此文件中。
## [2.2.1] - 2026-03-10
### Changed
- guide page: 添加功能介绍
- 解析word 格式：从word复制的内容会自动识别并格式化
- Memo支持“置顶/星标”
- **UI 视觉增强**：重构“置顶”与“星标”样式。置顶采用蓝色“大头针”+侧边装饰条，星标采用琥珀金“勋章”+标题加粗，实现视觉快速区分。
- **智能粘贴 2.0**：新增 Word 字体深度扫描。自动识别黑体为二级标题（##），识别楷体为三级标题（###），并自动生成大纲。

### Fixed
- **格式纠偏**：修复了 Markdown 加粗符号内部多余空格的问题。
- **呼吸间距**：实现了加粗块后的自动补空逻辑，解决文字粘连。
- **强制断行**：修复了从 Word 粘贴时部分段落不换行的 Bug。
## [2.2.0] - 2026-03-10
### Added
- **智能素材抓取系统**：正式落地内容抓取与入库能力，支持微信公众号文章一键提取标题与正文。
- **微信排版适配**：针对 `mp.weixin.qq.com` 深度适配 DOM 结构，实现段落级拆分、自动小标题识别与 Markdown 化渲染。

### Fixed
- **排版粘连问题**：彻底修复了章节编号（如“01 / 02 / 03”）后正文粘连成行的问题，保证公文级分段效果。
- **尾部杂质过滤**：增强来源、编辑、二维码推广、往期推荐等尾部噪音识别与熔断逻辑，避免将工具栏与宣传信息写入素材库。

## [2.1.1] - 2026-03-10
### Fixed
- 修复了移动端部分 UI 元素的对齐误差。
- 进一步优化了首页热力图与侧边栏的响应式间距。
- 隐藏了移动端不必要的冗余展示逻辑。

## [2.1.0] - 2026-03-10
### Added
- **首页重构**：热力图实现居中展示，视觉重心更聚焦。
- **动态侧栏**：历史时间轴与使用手册改为右侧吸附式（Sticky Sidebar），支持长页面随动。
- **iOS 兼容性**：彻底修复了 iPhone 14 等 iOS 设备在解锁界面无法点击输入框的层级（z-index）Bug。

### Changed
- **沉浸式阅读**：移动端文章详情页排版大手术，取消左右留白，字号提升至 17px，行高 1.85，对标微信公众号阅读体验。
- **精简模式**：移动端默认隐藏大纲提炼、批注笔记、深度思考模块，确保阅读零干扰。

## [2.0.4] - 2026-03-10
### Fixed
- 修复了新建素材页面提交按钮失效、无法保存内容的严重 Bug。
- 修复了底部操作栏在低分辨率屏幕下被遮挡的问题。

### Changed
- **编辑器进化**：取消 EasyMDE 编辑器的固定高度限制，改为随内容自动增长的流式布局。
- **交互优化**：将“保存/取消”按钮重构为集成式吸附底栏，保持宽度与编辑器对齐，消除视觉僵硬感。

## [2.0.1] - 2026-03-10
### Added
- **深度溯源**：语录本增加“查看原文”功能。
- **精准定位**：实现点击摘抄后跳转原文并自动滚动至对应段落，支持关键词临时高亮显示。

## [2.0.0] - 2026-03-09
### Added
- **核心模块上线**：正式发布“随心记（Memos）”与“语录本（Quotes）”双核心功能。
- **数据看板**：引入 GitHub 风格的贡献热力图，直观展示素材积累进度。
- **安全锁定**：上线“开门见锁”沉浸式锁定系统，保障公文素材私密性。
- **公文适配**：支持 A4 纸感阅读模式及 Word 格式导出。
